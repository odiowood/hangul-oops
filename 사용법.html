한글 Oops

한/영 전환키 안 누르고 잘못 친 글자, 지우지 말고 단축키 하나로 고치세요.

이게 뭔가요?

한영키를 깜빡하고 그냥 쳐서 dkssud 같은 오타가 났을 때, 전부 지우고 다시 칠 필요 없이 그 글자를 드래그로 선택하고 단축키만 누르면 제대로 고쳐줍니다.

dkssud→안녕
dkssudgktpdy→안녕하세요
ㅗ디ㅣㅐ→hello
WkaQhd→짬뽕
dkssud ㅗ디ㅣㅐ→안녕 hello

방향(한→영 / 영→한)은 자동으로 감지합니다. 한글·영문이 섞여 있어도 구간별로 알아서 서로 바꿔주고, Caps Lock이 켜진 채 친 경우까지 잡아줍니다.

1 Hammerspoon 설치

hammerspoon.org에서 내려받아 Hammerspoon.app을 응용 프로그램 폴더로 옮기고 한 번 실행하세요. (이 도구가 돌아가는 무료 엔진입니다.)

2 한글 Oops 설치

이 설명서가 들어있던 폴더의 install.command 파일을 더블클릭 하세요. 스푼 복사와 설정을 자동으로 해줍니다.

⚠️ macOS가 “Apple이 install.command에 악성 코드가 없는지 확인할 수 없습니다”라며 막으면(서명 안 된 스크립트라 정상): 시스템 설정 → 개인정보 보호 및 보안을 열고 아래로 스크롤하면 “install.command 이(가) 차단되었습니다” 옆에 [그래도 열기] 버튼이 있어요. → 누르고 → 암호/Touch ID → 다시 뜨는 창에서 [열기].
(최신 macOS에서는 예전의 “우클릭 → 열기”가 더 이상 동작하지 않습니다.)
💡 더 매끄러운 방법: 터미널을 열어 아래 한 줄을 붙여넣고 Enter 하면 경고 없이 설치됩니다.
curl -fsSL https://raw.githubusercontent.com/odiowood/hangul-oops/main/install.sh | bash

3 손쉬운 사용 권한 켜기

키 입력을 대신 눌러주려면 권한이 한 번 필요합니다. 시스템 설정 → 개인정보 보호 및 보안 → 손쉬운 사용에서 Hammerspoon을 켜주세요.

권한을 켠 뒤에는 Hammerspoon을 한 번 재시작하면 확실히 인식됩니다.

🖱 사용법

아무 앱에서나 잘못 친 텍스트를 드래그로 선택한 뒤 → ⌘ ⇧ ;

⌨️ 단축키 바꾸기 (코드 편집 없이!)

화면 오른쪽 위 메뉴바에 생긴 가A 아이콘을 클릭하세요.

  1. 단축키 바꾸기 — 미리 준비된 조합 중에서 고르기
  2. 직접 지정 (다음에 누르는 키로) — 클릭하고 원하는 키 조합을 (예: ⌘⇧H) 그냥 누르면 바로 지정됩니다

바꾼 단축키는 저장되어 컴퓨터를 껐다 켜도 그대로 유지됩니다.

❓ 잘 안 될 때

단축키를 눌러도 반응이 없어요 → 3번 손쉬운 사용 권한을 켰는지, 켠 뒤 Hammerspoon을 재시작했는지 확인하세요.

단축키가 다른 기능과 겹쳐요 → 위 메뉴바에서 다른 조합으로 바꾸세요.

🔔 업데이트 알림

새 버전이 나오면 자동으로 알려드립니다. 설치해두면 하루에 한 번 조용히 최신 버전을 확인하고, 업데이트가 있을 때만 알림을 띄웁니다. (알림을 클릭하면 다운로드 페이지가 열려요.)

지금 바로 확인하고 싶으면 메뉴바 가A → 업데이트 확인… 을 누르세요.

🗑 그만 쓰고 싶을 때 (제거)

설치할 때 받았던 폴더의 uninstall.command 파일을 우클릭 → 열기 하세요. 단축키·메뉴바 아이콘이 사라지고 관련 설정이 깔끔하게 제거됩니다. (기존에 쓰던 다른 Hammerspoon 설정은 그대로 둡니다.)

Hammerspoon 앱까지 지우려면 응용 프로그램에서 Hammerspoon을 휴지통으로 옮기면 됩니다. 다시 쓰고 싶으면 install.command를 실행하면 돼요.
